ALASAN UTAMA: INI ADALAH TASK REGRESSION, BUKAN CLASSIFICATION!

PERBEDAAN TASK:
===============

REGRESSION (Prediksi Nilai Kontinyu - Yang Kita Lakukan):
  - Output: Angka continuous (0-20)
  - Contoh: Prediksi nilai siswa 15.7
  - Metrics: R2, RMSE, MAE, MAPE
  - Apa metrics ini?
    * R2: Berapa % variance yang dijelaskan model (0-1)
    * RMSE: Average error dengan penalti untuk error besar
    * MAE: Average error sederhana

CLASSIFICATION (Prediksi Kategori - Beda Kasus):
  - Output: Kategori diskrit (Pass/Fail, A/B/C/D)
  - Contoh: Prediksi apakah siswa LULUS atau TIDAK
  - Metrics: Accuracy, Precision, Recall, F1-Score
  - Apa metrics ini?
    * Accuracy: % prediksi yang benar
    * Precision: Dari yang diprediksi LULUS, berapa % benar LULUS?
    * Recall: Dari yang LULUS sebenarnya, berapa % yang ter-predict?
    * F1-Score: Harmonic mean dari Precision & Recall

JADI TIDAK BISA PAKAI CLASSIFICATION METRICS UNTUK REGRESSION!

PENJELASAN MASING-MASING REGRESSION METRICS:
===============================================

R2 (Coefficient of Determination):
  Formula: R² = 1 - (SS_residual / SS_total)
  Range: 0 to 1 (semakin tinggi semakin baik)
  Interpretasi:
    - R² = 0.8466 = Model menjelaskan 84.66% dari variasi nilai siswa
    - Semakin dekat ke 1.0, semakin baik
    - Menunjukkan "seberapa baik model fit dengan data"
  
  Kenapa pakai ini?
    ✓ Mudah dipahami (persentase variance)
    ✓ Scale-independent (0-1 universal)
    ✓ Industry standard untuk regression
    ✓ Bisa compare antar model berbeda

RMSE (Root Mean Square Error):
  Formula: RMSE = sqrt(mean((y_actual - y_predicted)^2))
  Unit: Same as target (dalam skala 0-20)
  Interpretasi:
    - RMSE = 1.2233 = Average error ±1.22 poin
    - Pada skala 0-20, ini = ~6% error rate
    - PENTING: Error besar di-penalize lebih keras
  
  Kapan pakai?
    ✓ Ketika error besar sangat merugikan (quadratic penalty)
    ✓ Sensitive terhadap outliers (lebih ketat)
    ✓ Contoh: Prediksi dosis obat (error 10x lebih serius dari error 1)
  
  Kenapa RMSE?
    ✓ Standard untuk regression
    ✓ Lebih sensitif mendeteksi error besar
    ✓ Cocok untuk academic/production reporting

MAE (Mean Absolute Error):
  Formula: MAE = mean(|y_actual - y_predicted|)
  Unit: Same as target (dalam skala 0-20)
  Interpretasi:
    - MAE = 0.7108 = Average error ±0.71 poin
    - Pada skala 0-20, ini = ~3.6% error rate
    - Tanpa penalti extra untuk error besar (linear)
  
  Kapan pakai?
    ✓ Ketika semua error sama pentingnya (linear)
    ✓ Robust terhadap outliers (lebih toleran)
    ✓ Contoh: Prediksi tinggi badan (5cm off = 5cm, tidak ada extra penalty)
  
  Kenapa MAE?
    ✓ Mudah diinterpretasi (langsung rata-rata error)
    ✓ Tidak ada skewness dari outliers
    ✓ Berguna untuk practical understanding

HUBUNGAN: RMSE >= MAE (SELALU!)
  Random Forest: RMSE 1.2233 >= MAE 0.7108
  ✓ Menunjukkan ada beberapa error yang cukup besar
  ✓ Jika RMSE >> MAE = banyak outliers/error besar
  ✓ Jika RMSE ≈ MAE = error tersebar uniform

APA YANG TIDAK KAMI GUNAKAN?
=============================

Accuracy:
  - Hanya untuk classification (prediksi kategori)
  - Untuk regression: Tidak applicable
  - Contoh: Accuracy 95% pada regression = nonsense

Precision & Recall:
  - Hanya untuk classification
  - Berguna untuk: Prediksi Pass/Fail, Lulus/Tidak Lulus
  - Untuk prediksi nilai continuous: Tidak relevan

F1-Score:
  - Harmonic mean dari Precision & Recall
  - Hanya untuk classification
  - Untuk regression: Tidak bisa dihitung

Confusion Matrix:
  - Hanya untuk classification
  - Berguna untuk: Show TP, TN, FP, FN
  - Untuk regression: Tidak applicable
